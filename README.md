## 🚀 How to Use This File

### In Google Earth / Google My Maps
1. Download the `india-all-states.kml` file from this repository.
2. Open Google Earth or Go to Google My Maps.
3. Click **Import** and select the file. 
4. You will see all states appear in your sidebar as separate, toggleable layers.

### In Your Own App (Code Integration)
This file is ready for web and mobile development. It works seamlessly with:
- Google Maps JavaScript API
- Leaflet.js (using the Leaflet-OMV or KML plugins)
- Mapbox GL JS
